# `command_palette_match_color = rgba(0.95, 0.6, 0.25, 1.0)`

{{since('nightly')}}

Specifies the color used to highlight the portions of each command palette
entry that match the text you have typed into the filter
([ActivateCommandPalette](../keyassignment/ActivateCommandPalette.md)).

The highlight is applied both in the default fuzzy matching mode and in the
exact matching mode; see
[command_palette_match_mode](command_palette_match_mode.md).

See also [command_palette_fg_color](command_palette_fg_color.md) and
[command_palette_key_color](command_palette_key_color.md).
