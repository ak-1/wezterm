# `command_palette_key_color = rgba(0.5, 0.65, 0.95, 1.0)`

{{since('nightly')}}

Specifies the color used to render the keyboard shortcut / key assignment
shown alongside each entry in the command palette
([ActivateCommandPalette](../keyassignment/ActivateCommandPalette.md)).

This color is also used for the match-mode indicator shown on the input line.

See also [command_palette_fg_color](command_palette_fg_color.md),
[command_palette_bg_color](command_palette_bg_color.md) and
[command_palette_match_color](command_palette_match_color.md).
